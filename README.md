## PROYECTO COTIZADOR

El presente proyecto se encargará de generar una cotización para un cliente con tan solo ingresar algunos datos iniciales, despues el cotizador se encargará de realizar los calculos de forma automática y presentara el precio final al asegurado, permitiendo repetir el proceso de forma continua con diferentes aseguarados hasta los que se necesiten.

### PARTES DEL PROBLEMA

- Cotizacion lenta, debido a que es manual
- Demasiado tiempo de Entrega
- Perdida de Clientes

### CONSIDERACIONES PARA MEJORAR

- Mejorar las Seguridad del cotizador al solicitar usuario y contraseña
- Que al terminar la cotización permita elegir entre presentar los datos o enviarlos por correo o imprimir la cotización
- Que permita guardar las cotizacion hechas para consultas futuras
